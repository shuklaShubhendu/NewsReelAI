# News Reel AI 🎥✨

**Generate engaging news videos automatically with the power of AI!** 🚀

News Reel AI is a Python-based pipeline that transforms trending news articles into captivating videos using Stable Diffusion and AI-driven scripting. Perfect for content creators, journalists, or media enthusiasts looking to automate dynamic news content creation. 📺📰

---

## Features 🌟

- 🗞️ **News Fetching**: Pulls trending articles from RSS feeds, with fallback to full article scraping.
- ✍️ **AI Script Generation**: Crafts structured video scripts using advanced AI models.
- 🎬 **Video Creation**: Produces videos with Stable Diffusion-powered visuals and narration.
- 📂 **Customizable Output**: Saves videos with clean, timestamped filenames.
- 🛡️ **Error Handling**: Robust pipeline with clear error feedback.
- 🧩 **Modular Design**: Easily extendable with separate modules for fetching, scripting, and video creation.

---

## How It Works ⚙️

1. **Fetch Trending News** 🗳️: Grabs the top 5 trending articles from RSS feeds, selecting one for processing. Uses RSS summaries or scrapes full articles if needed.
2. **Generate Script** 📝: AI processes the article’s title and content to create a video-ready script with scenes.
3. **Create Video** 🎞️: Transforms the script into a video with Stable Diffusion visuals, saved to the `output/` folder.
4. **Output** 💾: Videos are stored with sanitized filenames, and execution details are logged.

## Prerequisites 🛠️

To run News Reel AI, you’ll need:

- 🐍 **Python**: Version 3.8 or higher
- 📦 **Dependencies**: Listed in [Dependencies](#dependencies)
- 🔑 **API Keys**: Required for some AI models (e.g., Stable Diffusion)
- 🖼️ **Stable Diffusion**: Configured locally or via a cloud service
- 🎥 **FFmpeg**: For video processing
- 📺 **Media Player**: For playing generated videos (e.g., VLC)

---

## Installation ⚡️

1. **Clone the Repository** 📂:
   ```bash
   git clone https://github.com/your-username/news-reel-ai.git
   cd news-reel-ai
   ```

2. **Set Up a Virtual Environment** 🌐:
   ```bash
   python -m venv venv
   source venv/bin/activate  # On Windows: venv\Scripts\activate
   ```

3. **Install Dependencies** 📦:
   ```bash
   pip install -r requirements.txt
   ```

4. **Configure Settings** ⚙️:
   - Edit `config.py` to set the output folder and API keys:
     ```python
     OUTPUT_FOLDER = "output/"
     # Add API keys or model configurations
     ```

5. **Install FFmpeg** 🎬:
   - Download from [ffmpeg.org](https://ffmpeg.org/download.html) or install via:
     ```bash
     # Ubuntu
     sudo apt-get install ffmpeg
     # macOS
     brew install ffmpeg
     ```

## Usage 🚀

1. **Run the Pipeline**:
   ```bash
   python main.py
   ```

2. **What Happens**:
   - Fetches trending news 🗞️
   - Generates a script 📝
   - Creates a video 🎥 and saves it to `output/` (e.g., `Article_Title_20250531_011623.mp4`)
   - Logs execution time and status
